Space Explorer Club Website - Project Documentation

Theme of the Website
The theme of the website is "Space Explorer Club". It is designed as a community hub for students and space enthusiasts. The aesthetic uses a deep space palette (dark blues and blacks) with stellar imagery and gold/white typography to create an immersive "galactic" feel.

Pages Created
The website consists of four distinct pages:
1.Home (`index.html`): The entry point, featuring a hero section and a welcome message.
2.Missions (`missions.html`): A showcase of active missions using a responsive flex-based grid.
3.About Us (`about.html`): Information about the club's history, mission, and vision.
4.Join Us (`join.html`): A membership application form for new explorers.

HTML Tags Used (with Description)
- `<html>`: The root element that wraps all content on the page.
- `<head>`: Contains metadata, such as character set and the link to the CSS file.
- `<body>`: Contains all the visible content of the page.
- `<header>`: Used for the top navigation bar and logo.
- `<nav>`: Specifically used for navigation links and menus.
- `<ul>` & `<li>`: Used for creating the list of navigation links.
- `<a>`: Anchor tags used to link between pages.
- `<h1>`, `<h2>`, `<h3>`: Heading tags for titles of varying importance.
- `<main>`: Specifies the main content area of the webpage.
- `<section>`: Groups related content together logically.
- `<div>`: A generic container used for layout and styling (Flexbox container).
- `<p>`: Paragraph tags for text content.
- `<img>`: Used to embed images from external URLs.
- `<form>`: Wraps the input elements for user applications.
- `<label>`: Provides text descriptions for form inputs.
- `<input>` & `<textarea>`: Used for user data entry.
- `<button>`: Triggers the submission of the application form.
- `<footer>`: Used for the copyright information at the bottom of the page.

CSS Properties Used
The website's layout is built entirely using Flexbox.
- `display: flex`: Activates the Flexbox layout model for containers.
- `flex-direction`: Sets the direction of the flex items (e.g., `column` for stacking).
- `justify-content`: Aligns items along the main axis (e.g., `center`, `space-between`).
- `align-items`: Aligns items along the cross axis (e.g., `center`).
- `flex-wrap`: Allows items to wrap onto the next line for responsiveness.
- `gap`: Adds spacing between flex items.
- `margin` & `padding`: Controls white space around and inside elements.
- `background-color` & `color`: Defines the color of elements and text.
- `font-family`: Sets the typography for the website.
- `border-radius`: Adds rounded corners to cards and buttons.
- `box-shadow`: Adds depth to elements with subtle shadows.
- `transition`: Used for smooth hover effects on links and buttons.
- `z-index`: Ensures the sticky header stays on top of other content.
